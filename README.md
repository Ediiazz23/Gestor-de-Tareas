📝 Gestor de Tareas CLI

<p align="center">
  <img src="https://upload.wikimedia.org/wikipedia/commons/6/6a/JavaScript-logo.png" width="100" alt="JavaScript Logo">
</p>Aplicación de línea de comandos desarrollada en JavaScript usando Node.js.
Permite gestionar tareas de forma simple desde la terminal.

🚀 Funcionalidades

- Agregar tareas
- Listar tareas
- Marcar tareas como completadas
- Eliminar tareas
- Guardado automático en archivo JSON

🛠️ Tecnologías

- JavaScript
- Node.js
- Módulos nativos: "fs", "readline"

📦 Instalación

1. Clonar el repositorio:

git clone https://github.com/Ediiazz23/Gestor-de-Tareas/blob/main/app.js

2. Entrar en el proyecto:

cd TU_REPO

3. Ejecutar:

node app.js

📂 Estructura

📁 proyecto
 ├── app.js
 └── tareas.json

💻 Uso

Al ejecutar el programa se mostrará un menú interactivo:

1. Agregar tarea
2. Listar tareas
3. Completar tarea
4. Eliminar tarea
5. Salir

Seleccioná una opción ingresando el número correspondiente.

🧠 Notas

- Las tareas se guardan automáticamente en "tareas.json"
- No requiere instalación de dependencias externas
- Compatible con entornos como Termux
